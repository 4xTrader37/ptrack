# **App Name**: Perfume Profit Tracker

## Core Features:

- Sales Tracking: Record individual sales with customer details, perfume name(s), quantity, and price in PKR.
- Investment Tracking: Log investments, investor names, amounts in PKR, source (external or self-investment), and items purchased.
- Automated Calculation of Profit/Loss: Automatically compute total sales, investments, profit, loss, and profit/loss ratio in PKR based on selected time range.
- Inventory Management: Manage product inventory with cost price, selling price, quantity in stock; automatically reduce stock upon sales.
- Date-Based Filtering: Enable filtering of sales, investments, and profit/loss calculations by date range (Today, Last 7 Days, Last Month, Custom).
- Report Generation: Generates clear reports (PDF, Excel) containing sales data, investments, product name, sales amount, selling price and summaries within a specified time range.
- Profit/Loss Forecasting: Uses historical data to forecast future profitability, aiding in inventory purchasing. Tool usage incorporates historical market sales trends and perfumery demand as influencing factor to generate a forecast report for investments and profit/loss values

## Style Guidelines:

- Primary color: Rich purple (#6A4EA1), evoking sophistication and luxury associated with perfumes.
- Background color: Very light lavender (#F4F0F9), provides a soft, clean backdrop.
- Accent color: Soft rose (#A14E84), complements the primary and enhances visual interest.
- Body font: 'PT Sans', a clear, readable, humanist sans-serif.
- Headline font: 'Playfair', a stylish serif with high contrast lines.
- Use elegant, minimalist icons to represent data and navigation elements.
- Subtle transitions and animations to provide feedback and enhance user experience.
